Terminal con el que podemos interactuar con el sistema en un lenguaje natural con la ayuda de ChatGPT.

Para utilizarlo es imprescindible añadir nuestra llave (API key) que podemos obtener de platform.openai.com  secrets.json,
para ello desde el Visual Studio pulsar con el boton derecho encima del proyecto (Shell-GPT) y seleccionar Administrar Secretros de Usuario,
se abrira secrects.json y añadirmos nuestra llave en el campo "API-Key"
