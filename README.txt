Terminal con el que podemos interactuar con el sistema en un lenguaje natural con la ayuda de ChatGPT.

Para utilizarlo es imprescindible añadir nuestra llave (API key) que podemos obtener de platform.openai.com.
Para ello desde el Visual Studio pulsamos con el boton derecho encima del proyecto (Shell-GPT) y seleccionamos Administrar Secretros de Usuario,
se abrira el archivo secrects.json, añadirmos nuestra llave en el campo "API-Key" y listo.
